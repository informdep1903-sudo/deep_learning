## 2. ЦЕЛЬ И ЗАДАЧИ ДИСЦИПЛИНЫ

**Цель:** Сформировать у обучающихся системные компетенции в области архитектур, методов обучения и практического применения глубоких нейронных сетей для решения задач профессиональной деятельности в роли ML-инженера.

**Задачи дисциплины:**

- Изучить фундаментальные концепции нейронных сетей и глубокого обучения, включая математические основы и алгоритмы оптимизации;
- Освоить архитектуры глубоких нейросетей (MLP, CNN, RNN, трансформеры) и методы их обучения;
- Сформировать навыки реализации и настройки моделей глубокого обучения с использованием современных фреймворков (PyTorch, TensorFlow);
- Развить способность адаптировать и разрабатывать нейросетевые архитектуры для решения прикладных задач.

## 6. СИСТЕМА ОЦЕНКИ РЕЗУЛЬТАТОВ ОБУЧЕНИЯ

Оценка сформированности компетенций проводится в соответствии с дескрипторами, описанными ниже.

**Краткое описание системы оценивания:**

- **Текущий контроль:** защита лабораторных работ (каждая работа оценивается от 0 до 5 баллов). Критерии: корректность реализации, глубина анализа результатов (включая анализ сходимости для MF-3.1), качество кода и оформления;
- **Рубежный контроль:** коллоквиумы по теоретическому материалу модулей (проверка знания архитектур, математических основ, методов оптимизации).
- **Промежуточная аттестация (экзамен):** состоит из двух частей:
  1. **Теоретическая часть:** устный ответ на вопросы по билетам (проверка индикаторов DL-1.1 — DL-1.5, DL-1.9, а также MF-3.1 и PL-1.2).
  2. **Практическая часть:** защита итогового проекта (решение сквозной задачи с использованием глубокого обучения).
