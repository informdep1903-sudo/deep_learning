**1 ЗАДАНИЕ ДЛЯ СТУДЕНТОВ**

**Тема:** Генерация текста с помощью RNN (символьный уровень)

**Цель работы:**

- Освоить принципы работы RNN для задач генерации последовательностей.
- Научиться подготавливать текстовые данные (создание алфавита, кодирование символов).
- Реализовать модель LSTM для предсказания следующего символа.
- Обучить модель на текстовом корпусе.
- Реализовать функцию генерации текста с использованием sampling с температурой.
- Проанализировать качество сгенерированных текстов.

**Задачи:**

1. Выбрать текстовый корпус для обучения. Например, произведения Л.Н. Толстого — файл “war\_and\_peace.txt”.
2. Выполнить предобработку текста:
- Создать алфавит (уникальные символы из текста).
- Построить словари для отображений “символ → индекс” и “индекс → символ”.
- Преобразовать текст в последовательность индексов.
- Сформировать обучающие последовательности (вход - последовательность символов, выход - следующий символ).
3. Реализовать модель RNN/LSTM в PyTorch:
- Архитектура: слой встраивания (Embedding), LSTM-слой (1–2 слоя), полносвязный слой с Softmax.
- Функция потерь: CrossEntropyLoss.
- Оптимизатор: Adam.
4. Обучить модель:
- Настроить гиперпараметры (количество эпох, размер батча, длина последовательности, скорость обучения).
- Отслеживать потери на обучающей выборке.
- Построить график потерь.
5. Реализовать функцию генерации текста:
- На вход подаётся начальная строка (seed).
- Модель предсказывает следующий символ (с использованием функции выборки sampling с температурой).
- Генерация продолжается до достижения желаемой длины выходной последовательности.
- Параметр температуры регулирует "креативность" (T < 1 — более детерминированный, T > 1 — более случайный).
6. Сгенерировать несколько примеров текста с разными температурами и проанализировать качество:
- При T = 0.5, T = 1.0, T = 1.5.
- Сравнить осмысленность, грамматику, стиль.
7. Сравнить качество генерации с использованием LSTM и GRU (обучить обе модели и сравнить).
8. Исследовать влияние длины обучающей последовательности (seq\_length = 50, 100, 200) на качество генерации.
9. Подготовить отчет.
