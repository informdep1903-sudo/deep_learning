# Глубокое обучение

Рабочая программа дисциплины (РПД) курса **«Глубокое обучение»** в рамках программы ДПО.  
Преподавание современного ИИ на топ-уровне: системное видение.

> **Группа 9** · Репозиторий проекта РПД

## Цель дисциплины

Сформировать у обучающихся системные компетенции в области архитектур, методов обучения и практического применения глубоких нейронных сетей для решения задач профессиональной деятельности в роли ML-инженера.

## Задачи дисциплины

1. Изучить фундаментальные концепции нейронных сетей и глубокого обучения, включая математические основы и алгоритмы оптимизации.
2. Освоить архитектуры глубоких нейросетей (MLP, CNN, RNN, трансформеры) и методы их обучения.
3. Сформировать навыки реализации и настройки моделей глубокого обучения с использованием современных фреймворков (PyTorch, TensorFlow).
4. Развить способность адаптировать и разрабатывать нейросетевые архитектуры для решения прикладных задач.
