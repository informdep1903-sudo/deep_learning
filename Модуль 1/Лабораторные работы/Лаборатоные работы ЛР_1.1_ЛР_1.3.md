## ЛР 1.1. Реализация многослойного перцептрона (MLP) на NumPy

- **Цель:** Освоить базовые принципы построения и обучения нейронных сетей без использования готовых фреймворков.
- **Задачи:**
  1. Реализовать класс MLP с возможностью задания количества слоёв, нейронов и функций активации.
  2. Реализовать прямое распространение (forward pass) и обратное распространение ошибки (backpropagation) вручную с использованием NumPy.
  3. Обучить модель на датасете MNIST для задачи классификации.
  4. Визуализировать процесс обучения (графики потерь и точности).
- **Технологии:** Python, NumPy, Matplotlib.
- **Компетенции:** DL-1.1 (Б/С), DL-1.2 (С), PL-1.2 (С).

## ЛР 1.2. Эксперименты с оптимизаторами и регуляризацией

- **Цель:** Исследовать влияние различных методов оптимизации и регуляризации на процесс обучения и качество модели.
- **Задачи:**
  1. Реализовать на базе MLP из ЛР 1.1 следующие оптимизаторы: SGD, SGD с моментом, Adam, RMSProp.
  2. Сравнить скорость сходимости и итоговую точность для каждого оптимизатора на MNIST.
  3. Реализовать механизмы регуляризации: L1, L2, Dropout.
  4. Провести эксперименты по влиянию регуляризации на переобучение (сравнить графики на тренировочной и валидационной выборках).
  5. Построить сравнительные графики и сделать выводы.
- **Технологии:** Python, NumPy, Matplotlib.
- **Компетенции:** DL-1.1 (С), MF-3.1 (С), PL-1.2 (С).
