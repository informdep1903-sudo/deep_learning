# Фонд оценочных средств по дисциплине «Глубокое обучение»

> **Статус:** Предфинальная версия. Требуется наполнение ЛР : Модуль 1, Модуль 4

Этот репозиторий содержит фонд оценочных средств (ФОС) по дисциплине «Глубокое обучение». Здесь объединены рабочая программа дисциплины, модель измерения результатов обучения, контрольно-измерительные материалы (КИМ), критерии оценивания, методические рекомендации и используемые образовательные ресурсы. Результаты обучения связываются с компетентностно-ролевой моделью в области искусственного интеллекта (КРМ версии 3.0).

## 1. О дисциплине

**Цель дисциплины.** Сформировать у обучающихся системные компетенции в области архитектур, методов обучения и практического применения глубоких нейронных сетей для решения задач профессиональной деятельности в роли ML-инженера.

**Основные задачи.**

- Изучить фундаментальные концепции нейронных сетей и глубокого обучения, включая математические основы и алгоритмы оптимизации.
- Освоить архитектуры глубоких нейросетей (MLP, CNN, RNN, трансформеры) и методы их обучения.
- Сформировать навыки реализации и настройки моделей глубокого обучения с использованием современных фреймворков (PyTorch, TensorFlow).
- Развить способность адаптировать и разрабатывать нейросетевые архитектуры для решения прикладных задач.

**Структура и содержание.** 

Дисциплина состоит из **6 модулей**:

1. Основы глубокого обучения.
2. Основы свёрточных нейронных сетей.
3. Инструментарий глубокого обучения (PyTorch, TensorFlow, Keras).
4. Современные архитектуры и применение CNN.
5. Рекуррентные нейронные сети (RNN).
6. Трансформеры.

Обучение включает лекции (36 ч), лабораторные работы (54 ч), самостоятельную работу (90 ч) и завершается экзаменом с защитой итогового проекта.

**Пререквизиты и связи с другими дисциплинами.**

 Для успешного освоения необходимы знания по теории вероятностей и математической статистике, классическому машинному обучению, программированию на Python. Дисциплина является базой для последующих курсов: генеративные нейронные сети, обработка естественного языка, рекомендательные и поисковые системы.

**Планируемые результаты обучения.** 

По завершении дисциплины обучающийся сможет:

- Объяснять и применять математические основы нейронных сетей (градиенты, обратное распространение, оптимизация, регуляризация).
- Реализовывать и настраивать нейронные как базовых архитектур (MLP, SOM, RBF), так и глубокие нейронные сети современных архитектур (CNN, RNN, LSTM, GRU, Трансформеры).
- Использовать фреймворки PyTorch и TensorFlow/Keras для обучения, оценки и развёртывания моделей.
- Проводить эксперименты с гиперпараметрами, анализировать сходимость и интерпретировать результаты.
- Адаптировать предобученные модели (трансферное обучение) и применять их к новым задачам.

Полное описание приведено в [РПД](DOCS/РПД_Глубокое%20обучение.md).

## 4. Итоговая оценка

Общая сумма баллов за все формы контроля — **100**. Распределение:

| Форма контроля | Макс. балл |
|----------------|:----------:|
| Лабораторные работы (суммарно) | 40 |
| Рубежный контроль (коллоквиумы) | 20 |
| Итоговый проект | 20 |
| Экзамен (теоретическая часть) | 20 |
| **Итого** | **100** |

**Условия допуска и зачёта результатов:**

- К экзамену допускается обучающийся, представивший все обязательные лабораторные артефакты и комплексный кейс (итоговый проект).
- Положительный итог — не менее **60 баллов из 100** и не менее **50%** доступных баллов по экзамену (т.е. ≥ 10 баллов из 20).
- Средний уровень индикатора подтверждается не только суммой баллов, но и наличием обязательного артефакта и выполнением дескриптора по матрице оценивания.
- Повторная попытка выполняется на новом варианте данных или с изменённым ограничением, чтобы проверить перенос способности.

**Перевод баллов в пятибалльную шкалу:**

| 100-балльная оценка | 5-балльная оценка | Уровень освоения |
|:-------------------:|:-----------------:|------------------|
| 86–100 | 5 (отлично) | Средний уровень освоен полностью |
| 70–85 | 4 (хорошо) | Средний уровень освоен в достаточной степени |
| 60–69 | 3 (удовлетворительно) | Средний уровень освоен частично |
| 0–59 | 2 (неудовлетворительно) | Средний уровень не освоен |
